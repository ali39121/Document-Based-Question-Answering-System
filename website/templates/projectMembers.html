Mohammad Ali

Hey! I'm Mohammad Ali, and I'm excited to share our Document-Based Question Answering System. This project tackles the challenge of finding answers within documents for a given question. We built it using the Langchain framework, which provides a powerful foundation for building AI applications. At the core of the system lies DistillBERT, a large language model (LLM) pre-trained for understanding and generating human language. To represent the documents and questions in a way the LLM can understand, we employed embeddings from Hugging Face, a popular library for natural language processing. For the user interface, we used Flask.py, a Python web framework, along with HTML and CSS to create a user-friendly experience. The entire project is available on GitHub, so feel free to explore and learn more about how it works!
